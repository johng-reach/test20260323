# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Environment

Python 3.12.3 virtual environment located at `base/`. Activate it before running any Python code:

```bash
source base/bin/activate
```

## Running Code

```bash
# Run the Python script
python base/hello.py

# Launch Jupyter to work with notebooks
jupyter notebook
```

## Structure

- `base/` — Python 3.12.3 venv; also contains `hello.py` and two `.ipynb` notebooks directly inside it
- `base/hello.py` — simple "Hello World" script
- `base/hello_world.ipynb` — Hello World Jupyter notebook
- `base/20260323 1510 Test.ipynb` — test notebook
